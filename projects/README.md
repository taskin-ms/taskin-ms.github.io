# PhD Project Materials

This folder contains materials for your ongoing PhD research project.

## Structure

- `phd-project.html` - Main project page
- `papers/` - Research papers and manuscripts
- `presentations/` - Slides and presentation materials
- `code/` - Code snippets and links to repositories
- `documentation/` - Project documentation and notes
- `images/` - Figures, diagrams, and images for the project page

## Usage

1. Add your project materials to the appropriate folders
2. Update `phd-project.html` with your project information
3. Link to this page from your main portfolio

## Accessing the Project Page

The project page will be accessible at: `https://taskin-ms.github.io/projects/phd-project.html`
