# Personal Portfolio Website

A clean, responsive one-page portfolio website built with HTML, CSS, and JavaScript.

## Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Animated sections with fade-in effects
- Modern gradient design
- Sections for About, Projects, and Contact

## Deployment to GitHub Pages

To deploy this website to GitHub Pages:

1. Create a new repository on GitHub named `<username>.github.io` (replace `<username>` with your GitHub username)

2. Add the remote repository:
   ```bash
   git remote add origin https://github.com/<username>/<username>.github.io.git
   ```

3. Add and commit your files:
   ```bash
   git add .
   git commit -m "Initial commit: Personal portfolio website"
   ```

4. Push to GitHub:
   ```bash
   git push -u origin main
   ```

5. Your website will be live at `https://<username>.github.io`

## Customization

Edit the following files to personalize your website:

- **index.html**: Update your name, tagline, about section, projects, and contact links
- **styles.css**: Customize colors by modifying the CSS variables in the `:root` section
- **script.js**: Add additional interactivity if needed

## Local Development

Simply open `index.html` in your web browser to view the website locally.
